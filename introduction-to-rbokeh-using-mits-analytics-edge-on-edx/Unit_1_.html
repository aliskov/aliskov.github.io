The course is fantastic, but let’s try to improve the visualizations a bit. Let’s install RBokeh from the amazing team at Anaconda.

install.packages("devtools")
devtools::install_github("ramnathv/htmlwidgets")
devtools::install_github("bokeh/rbokeh")

Now let’s recreate the plot from the course but in a better visual way.

library(rbokeh)
p = figure(width = 700) %>% 
  ly_points(GNI, FertilityRate,
            data = WHO, 
            color=Region, 
            hover=list(Country,GNI,FertilityRate,LifeExpectancy))
